# AI & Agent Engineering Practices

A distilled good-vs-bad practice reference for building software with LLMs and agents,
mined from the **AI Devs 4** (`s01e01`…`s05e05`) and **10xDevs 3** course material and a
sample of its Claude Code artifacts (skills, rules, prompts).

Each theme document is structured the same way: a short intro, then **## ✅ Good practices**
and **## ❌ Anti-patterns**, where every entry carries a bold title and a one-to-two-sentence
explanation. This mirrors the ✅/❌ guideline standard used by the QA-orchestration product in
this repo.

> ~750 curated, deduplicated practices across 13 themes.
